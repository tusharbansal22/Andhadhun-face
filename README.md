# Andhadhun-face


##### If Node is not installed
[Click here to install Node](https://nodejs.org/en/download)


##### Dependencies
```
npm i child_process
npm i qrcode
npm i http
npm i express
npm i python-shell
npm i node-webcam
npm i child_process
```

##### To Run
 ```
 node index.js
 ```
 
 If you have nodemon installed
 ```
 nodemon index.js
 ```
